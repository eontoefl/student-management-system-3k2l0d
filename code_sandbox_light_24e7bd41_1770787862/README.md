## 🎯 프로젝트 개요

### 목적
- **내벨업챌린지**와 **스라첨삭** 두 프로그램을 하나의 통합 시스템으로 관리
- 학생 중심 구조로 모든 정보를 한 곳에서 조회 및 관리
- 관리자가 학생의 전체 라이프사이클(신청 → 챌린지 → 첨삭 → 완료)을 추적

### 주요 특징
- ✅ **단일 학생 중심 UI**: 한 학생의 모든 정보를 한 곳에서 관리
- ✅ **5개 탭 구조**: 기본정보, 성적관리, 시험결과, 진행현황, 스라첨삭
- ✅ **자동 날짜 계산**: 챌린지 종료일, 스라첨삭 시작 가능일 자동 계산
- ✅ **통합 대시보드**: 전체/진행중/대기/완료 학생 수 실시간 표시
- ✅ **검색 기능**: 이름/연락처로 빠른 검색
- ✅ **반응형 디자인**: 모바일/태블릿/데스크톱 지원

## 🚀 주요 기능

### 1. 학생 관리
- **신규 학생 등록**
  - 기본 정보: 이름, 연락처 (자동 하이픈 포맷팅) 🆕
  - 챌린지 정보: 프로그램 선택(Fast/Standard), 시작일
  - 현재 성적: 개정전(0-120점) 또는 개정후(1-6 레벨) + **실시간 자동 계산** 🆕
  - 목표 성적: **섹션별 입력 또는 총점만 입력 선택 가능** 🆕
  - 기타 정보: 마지막 시험 날짜, 입금액 (자동 콤마 표시) 🆕

- **학생 목록 조회**
  - 테이블 형태로 전체 학생 표시
  - 이름, 연락처, 프로그램, 현재→목표, 챌린지 기간, 첨삭 상태, 신청 단계
  - 클릭 시 학생 상세 모달 열림

- **학생 상세 정보**
  - 5개 탭으로 모든 정보 조회
  - 각 탭에서 수정 가능 ⭐ NEW!

- **정보 수정 기능** ⭐ NEW!
  - **기본정보 수정**: 이름, 연락처, 프로그램, 시작일 수정 (종료일/시작 가능일 자동 재계산)
  - **성적 수정**: 현재 성적, 목표 성적, 마지막 시험일 수정
  - **진행현황 수정**: 신청 단계, 결제 정보, 마무리 체크 수정

### 2. 성적 관리
#### 현재 성적
- **개정전**: 각 섹션 0-30점, **총점 자동 계산** 🆕
- **개정후**: 각 섹션 1-6 레벨(0.5 단위), **총 레벨 자동 계산 (평균을 0.5 단위로 반올림)** 🆕

#### 목표 성적
- **입력 방식 선택** 🆕:
  - **섹션별 입력**: Reading, Listening, Speaking, Writing 각각 입력 (총 레벨 자동 계산)
  - **총점만 입력**: 목표 총 레벨만 입력 (예: 6.0)
- **개정후 레벨만 지원**: 1-6 레벨(0.5 단위)

#### 마지막 시험 날짜
- 정확한 날짜(YYYY-MM-DD) 또는 텍스트(예: 2026년 하반기)

### 3. 시험 결과 관리
- **시험 히스토리**: 최대 6회까지 기록
- **시험 결과 추가** ⭐ NEW!
  - 시험 회차(1-6차) 선택
  - 시험 날짜 입력
  - 섹션별 레벨 입력
  - 총 레벨 자동 계산
  - 중복 회차 방지
- **시험 결과 삭제** ⭐ NEW!
  - 각 시험 결과 카드에 삭제 버튼
  - 확인 후 삭제
- **각 시험 결과**:
  - 시험 회차(1-6차)
  - 시험 날짜
  - 섹션별 레벨: Reading, Listening, Speaking, Writing
  - 총 레벨: 자동 계산(평균을 0.5 단위로 반올림)

### 4. 진행 현황 관리
#### 신청 단계 (4단계)
1. 계약서
2. 택배
3. 액세스
4. 알림톡

**수정 기능** ⭐ NEW!: 각 단계 체크박스로 완료 여부 업데이트

#### 과제 달성률
- 과제 사이트 API 연동 예정 (현재 구현 대기)

#### 결제 정보
- 입금액 표시
- **수정 기능** ⭐ NEW!: 입금액 수정 가능

#### 마무리
- 후기 작성 여부
- 정산 완료 여부
- **수정 기능** ⭐ NEW!: 체크박스로 완료 여부 업데이트

### 5. 스라첨삭 배정
- **시작 가능일 자동 계산**
  - 챌린지 종료일 + 1일 후 첫 일요일
- **배정 기능** ⭐ **완전 구현!**
  - 학생 상세 모달 → [스라첨삭] 탭 → [배정하기] 클릭
  - 배정 가능한 모든 슬롯 표시
  - 선생님별 현재 담당 인원 표시
  - 시작일/종료일 자동 계산 (총 8회, 주 2회: 일/수)
  - 일정 겹침 자동 확인 및 조정
  - 부담이 적은 선생님 우선 표시
  - 4명째 배정 시 경고 표시
  - 배정 확인 모달로 최종 확인
- **미래 슬롯 정보** ⭐ NEW!
  - 모든 선생님 만석일 때 다음 가능 시점 표시
  - 어느 학생 완료 후 자리가 나는지 표시
- **배정 취소**
  - 배정 내역에서 삭제 가능
  - 학생 상태 자동 복구 (대기로 전환)

## 📖 사용 가이드

### 1. 신규 학생 등록하기
1. **[학생 등록]** 버튼 클릭
2. **기본 정보** 입력:
   - 이름
   - 연락처 (자동 하이픈 포맷팅: 010-XXXX-XXXX) 🆕
3. **챌린지 정보** 입력:
   - 프로그램 선택: Fast (4주) 또는 Standard (8주)
   - 챌린지 시작일 선택
4. **현재 성적** 입력:
   - 성적 타입 선택: 개정전(0-120점) 또는 개정후(1-6 레벨)
   - 각 섹션 점수/레벨 입력
   - **총점/총 레벨 자동 계산 확인** 🆕
5. **목표 성적** 입력:
   - **입력 방식 선택** 🆕:
     - 섹션별 입력: R/L/S/W 각각 입력 (총 레벨 자동 계산)
     - 총점만 입력: 목표 총 레벨만 입력 (예: 6.0)
6. **기타 정보** 입력 (선택사항):
   - 마지막 시험 날짜
   - 입금액 (자동 콤마 표시: 500,000) 🆕
7. **[등록하기]** 클릭
8. 시스템이 자동으로 계산:
   - 챌린지 종료일 = 시작일 + 4주(Fast) 또는 8주(Standard) - 1일
   - 스라첨삭 시작 가능일 = 종료일 + 1일 후 첫 일요일

### 2. 학생 정보 조회하기
1. **검색창**에서 이름 또는 연락처로 검색
2. **테이블**에서 학생 행 클릭
3. **학생 상세 모달**이 열림
4. **5개 탭**을 전환하며 정보 조회:
   - **[기본정보]**: 이름, 연락처, 프로그램, 챌린지 기간
   - **[성적관리]**: 현재 성적, 목표 성적, 마지막 시험
   - **[시험결과]**: 시험 히스토리 (1-6회)
   - **[진행현황]**: 신청 단계, 과제 달성률, 결제 정보, 마무리
   - **[스라첨삭]**: 시작 가능일, 배정 현황, 배정하기/취소

### 3. 스라첨삭 배정하기 ⭐ **완전 구현!**
1. 학생 상세 모달에서 **[스라첨삭]** 탭 선택
2. **[배정하기]** 버튼 클릭
3. 슬롯 선택 모달 열림:
   - 학생 이름과 시작 가능일 표시
   - 배정 가능한 모든 슬롯 목록 표시
   - 각 슬롯 정보:
     - 담당 선생님
     - 현재 담당 인원 (N/4명)
     - 시작일 (일요일)
     - 종료일 (수요일)
     - 총 8회 수업 안내
   - 3명 이상 담당 시: ⚠️ 4명째 배정 경고
   - 모든 선생님 만석 시: 미래 가능 시점 표시
4. 원하는 슬롯 클릭
5. 배정 확인 모달에서 최종 확인:
   - 학생 정보 (이름, 연락처, 프로그램, 챌린지 기간)
   - 배정 정보 (담당 선생님, 현재 담당, 시작/종료일)
   - 경고 메시지 (4명째인 경우)
6. **[배정 완료]** 버튼 클릭
7. 배정 완료 및 학생 상태 자동 업데이트

### 4. 배정 취소하기
1. 학생 상세 모달에서 **[스라첨삭]** 탭 선택
2. **[배정 취소]** 버튼 클릭
3. 확인 후 배정 취소
4. 학생 상태 자동으로 '대기'로 복구

### 5. 시험 결과 추가하기 ⭐ NEW!
1. 학생 상세 모달에서 **[시험결과]** 탭 선택
